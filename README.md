# Modelamiento de Procesos BPMN

Este proyecto presenta el modelamiento de procesos de negocio utilizando BPMN, enfocado en la optimización de flujos dentro de una organización de ventas.

## Objetivo
Analizar y mejorar procesos como:
- Gestión de pedidos
- Validación de pagos
- Despacho de productos
- Control de inventario

## Contenido
- Documento completo en PDF
- Modelamiento BPMN del sistema
- Definición de actores y procesos

## Tecnologías / Enfoque
- BPMN
- Análisis de procesos de negocio
- Mejora de flujos operacionales
